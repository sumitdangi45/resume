## 📋 Dost Ko Invite Karne Ke Liye (Read-Only Access)

### Option 1: Read-Only Access (Recommended)
Agar dost sirf code dekhe aur koi changes na kar sake:

#### Step 1: Go to Repository Settings
1. GitHub par apna repository kholo: https://github.com/sumitdangi45/resume
2. **Settings** tab par click karo (top right)
3. Left sidebar mein **Collaborators** par click karo

#### Step 2: Add Collaborator
1. **Add people** button par click karo
2. Apne dost ka GitHub username type karo
3. **Select a role** dropdown mein **Read** select karo
4. **Add [username] to this repository** par click karo

#### Step 3: Dost Ko Notification Milega
- GitHub notification milega
- Accept karna padega
- Phir wo repository dekh sakta hai but changes nahi kar sakta

## 📝 Alternative: Invite via Email

Agar dost ka GitHub account nahi hai:

1. Settings → Collaborators
2. **Add people** par click karo
3. Email address type karo
4. **Read** role select karo
5. Invite bhej do

Dost ko email milega, GitHub account banana padega, phir accept kar sakta hai.

## 🔄 Agar Baad Mein Permission Change Karna Ho

1. Settings → Collaborators
2. Dost ke naam ke paas dropdown mein role change karo
3. **Remove** button se remove kar sakta ho
