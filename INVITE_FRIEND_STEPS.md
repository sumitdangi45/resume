# 👥 Dost Ko Invite Karne Ke Liye - Step by Step

## 🎯 Goal
Dost ko read-only access dena (dekh sake, change na kar sake)

---

## 📱 Step 1: GitHub Settings Kholo

```
1. https://github.com/sumitdangi45/resume par jao
2. Top right mein "Settings" tab par click karo
3. Left sidebar mein "Collaborators" par click karo
```

**Screenshot path:**
```
Repository → Settings → Collaborators
```

---

## 👤 Step 2: Add People Button

```
"Add people" button par click karo
```

---

## 🔍 Step 3: Dost Ka Username Type Karo

```
Search box mein apne dost ka GitHub username type karo
Example: @dost_ka_username

Phir search results mein se select karo
```

---

## 🔐 Step 4: Role Select Karo

```
"Select a role" dropdown mein click karo
Options:
  • Read ← YE SELECT KARO (read-only)
  • Triage
  • Write
  • Maintain
  • Admin
```

**Read role = Sirf dekh sakta hai, changes nahi kar sakta** ✅

---

## ✅ Step 5: Invite Send Karo

```
"Add [username] to this repository" button par click karo
```

---

## 📧 Step 6: Dost Ko Notification

```
Dost ko GitHub notification milega:
"[Tumhara naam] invited you to collaborate on sumitdangi45/resume"

Dost ko accept karna padega
```

---

## 🎉 Done!

Ab dost:
- ✅ Repository dekh sakta hai
- ✅ Code read kar sakta hai
- ✅ Issues dekh sakta hai
- ❌ Changes nahi kar sakta
- ❌ Push nahi kar sakta
- ❌ Delete nahi kar sakta

## 💡 Tips

### Agar Dost Ko Write Access Dena Ho
1. Settings → Collaborators
2. Dost ke paas role change karo **Write** mein
3. Ab wo changes kar sakta hai

### Agar Remove Karna Ho
1. Settings → Collaborators
2. Dost ke paas **Remove** button par click karo

### Agar Email Se Invite Karna Ho
1. Add people par click karo
2. Email address type karo (GitHub username ki jagah)
3. Read role select karo
4. Invite send karo
